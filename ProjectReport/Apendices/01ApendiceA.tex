\section{Materiales adjuntos}
%-------------------------------------------------------------------
\label{ap1:intro}

Con la memoria se han entregado de manera adjunta en la carpeta de Google Drive dos archivos con extensión \textit{.apk} que corresponden a las dos aplicaciones móviles desarrolladas en este Trabajo de Fin de Grado. Puede realizarse la instalación de ambas para probar su funcionamiento sobre objetos reales.

El resto de materiales desarrollados pueden encontrarse en el repositorio de GitHub \url{https://github.com/celica02/RecyclingFinalDegreeProject}. Tanto el código de las aplicaciones como los modelos ya entrenados y las imágenes utilizadas para ello se encuentran así almacenadas en sus correspondientes carpetas. Toda la información del repositorio puede encontrarse en el archivo \texttt{Readme} de este mismo.
